Pycon

Pycon is a Node.js desktop application built with Electron, enabling efficient conference management. It utilizes Express for robust routing and SQLite for reliable data storage.

  • Node.js
  • Electron
  • Express
  • SQLite
  • Desktop Application
  • Conference Management

Features

  • Keep track of reference projects
  • Create reference project categories
  • Upload icon collections for those categories
  • Convert individual SVG icons into a sprite or a single SVG collection file
  • Rename or remove collection icons
  • Create your own icon project
  • Search for icons from reference projects
  • Add icons to your own projects
  • Keep track of these actions
  • Generate a single CSS file containing the SVG icon collection

Technology Stack

  • Node.js (runtime environment)
  • Electron (desktop framework)
  • Express (web framework)
  • SQLite (database management)
  • HTML/CSS/JavaScript (frontend)
  • SVG (icon graphics)
