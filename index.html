  • Express
  • SQLite

Features

  • Keep track of reference projects
  • Create reference project categories
  • Upload icon collections for those categories
  • Convert individual SVG icons into a sprite or a single SVG collection file

  • Generate a single CSS file containing the SVG icon collection
