Why a Dynamic JSON Form Builder?

Step 3: JavaScript Form Engine

generateEmptyData(config)

Generates an empty JSON object based on the form structure:

function generateEmptyData(config) {
  const obj = {};
  for (let key in config) {
    obj[key] = config[key].type === "subform" ? [] : "";
  }
  return obj;
}

renderForm(config, data, container, updateFn)
